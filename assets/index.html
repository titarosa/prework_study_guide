HTML

  • The head element contains information about the webpage.
    • We can use 6 sizes of headder h1 to h6.
    • We can use h1 size only one time.
  • The body element represents the visible content shown to the user.
  • The section element allow us to elegantly organize the content on our page based on the topics we have.

CSS

  • Controls how elements are arranged, including their size, spacing, and positioning on the webpage.
  • Allows for customization of text properties such as font size, font family, color, and alignment.
  • Adds visual effects like animations, transitions, and shadows to enhance the appearance of elements.

GIT

  • Tracks changes to files over time, allowing you to revert to previous versions and manage multiple versions of a project.
  • Facilitates collaboration by enabling multiple people to work on a project simultaneously without interfering with each other's work, through branching and merging.
  • Provides a robust system for backing up projects, making it easy to recover from mistakes or revert to a stable state.

JavaScript

  • Enables dynamic interactivity on websites, such as responding to user inputs, manipulating the DOM, and creating interactive forms.
  • Supports asynchronous operations like AJAX, allowing web pages to update dynamically without reloading.
  • Allows integration with various APIs and the use of powerful libraries and frameworks (e.g., React, Angular) to enhance web development.
